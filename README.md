# devenv

Development environment for SeaPork with Vagrant, K3S and friends.

## Getting started

It takes ~10 minutes to provision a server.

```
git clone https://gitlab.com/linux-polska/robot/seapork-devenv.git
cd devenv
vagrant up
```

## Getting in

To get into machine:

```
vagrant ssh
```

To re-provision:

```
vagrant provision
```
